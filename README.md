# Resume Parser Flask

## Project Description
A simple Resume Parser built using Flask and PDFPlumber. Users can upload a PDF resume, and the application extracts and displays the resume text.

## Features
- Upload PDF resumes
- Extract text from PDF files
- Display extracted text in the browser

## Technologies Used
- Python
- Flask
- PDFPlumber

## How to Run

1. Install dependencies:
   pip install flask pdfplumber

2. Run the application:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000
